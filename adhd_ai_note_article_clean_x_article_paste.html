プロンプトは呪文ではなく、「型」で考える

まず最初に、1つだけ覚えてください。

AIに頼む時は、次の5つがあると強いです。

  1. 役割
    例: 行動コーチ、編集者、秘書、要約者
  2. 状況
    例: 今やる気が出ない、返信に困っている、メモが散らかっている
  3. 目的
    例: 最初の10分で動きたい、角が立たない返信をしたい、記事構成を作りたい
  4. 条件
    例: 100字以内、3案、初心者向け、一問ずつ聞く
  5. 出力形式
    例: 箇条書き、手順、表、見出し構成

つまり、プロンプトの基本形はこうです。

あなたは[役割]です。
状況: [今の状況]
目的: [何をしたいか]
条件: [文字数、口調、制約]
出力: [どう出してほしいか]

これだけです。
別に呪文でも何でもない。
ただの依頼書です。

そして、ADHDの人に特におすすめなのは、条件に
「一度にたくさん言わず、1つずつ出して」
を入れることです。

これだけで、かなり使いやすくなります。

1. 朝の頭を起動するプロンプト

朝は、能力より前に、OSが立ち上がっていないことがあります。
この時に必要なのは、壮大な計画ではなく、最初の20分です。

あなたはADHD向けの行動コーチです。
今日やることは以下です。
- 14時までに請求書を送る
- 16時の会議準備
- 洗濯
- 母に連絡
今の状態: 眠い、だるい、やる気ゼロ、少し焦っている
目的: 最初の20分で確実に前進すること
条件:
- 最初にやることを1つだけ決める
- 5分単位で分解する
- 説明は短く
- 励ましは最後に一言だけ
出力: 手順

ポイントは、「全部やる方法」ではなく、「最初の20分だけ」にすることです。
ADHDは全体を見ると固まりやすい。
だから時間も範囲も、わざと小さく切る。

2. 頭の中のゴミ屋敷を整理するプロンプト

脳内が散らかりすぎて、何に困っているかすら分からない時があります。
この時は、綺麗にまとめようとしない。殴り書きでいい。

今から脳内をそのまま書きます。整理してください。
出力は以下の4つに分けてください。
1. 今日やること
2. 今週やること
3. 誰かに頼めること
4. 今は気にしなくていいこと

脳内メモ:
請求書まだ、返信2件、部屋やばい、髪切りたい、note書きたい、
税金こわい、眠い、なんか全部嫌、でも会議資料まだ、洗濯もある

これの良いところは、
「今は気にしなくていいこと」
を分けてくれる点です。

ADHDは、全部を同じ重さで抱えがちです。
AIに仕分けさせると、急に息がしやすくなる。

3. 返信地獄を終わらせるプロンプト

LINE、メール、Slack。
これだけでHPが削られる人は多いはずです。

以下のメッセージに返信したいです。
私の意図: 今週は断りたい、でも関係は悪くしたくない
条件:
- 角が立たない
- でも弱すぎない
- 80〜120字
- 日本語
- 候補を3つ
[相手のメッセージを貼る]

さらに強いのは、追加でこう言うことです。

相手が気分を害しにくい順に並べてください。
一番無難な案を1つ選び、その理由も短く書いてください。

これだけで、返信が「精神ダメージのある作文」から、
選択問題になります。
この差は大きい。

4. 先延ばしタスクに着手するプロンプト

ADHDは怠けているのではなく、
タスクが大きすぎて脳が拒否していることが多いです。

だから、やることは小さくする。異常なほど小さく。

私は今、確定申告の作業を先延ばししています。
理由は、面倒なのと、どこから始めるか分からないのと、少し怖いからです。
今から5分だけでできる最初の行動を1つに絞ってください。
その後に続く「次の1手」だけも書いてください。
条件:
- 説教しない
- 小さく分ける
- 今すぐできる内容にする

ここでのコツは、
「全部の手順」ではなく「最初の1手」だけを出させることです。

人は最初の1手が決まると、意外と2手目も行けます。
逆に、最初の1手が不明だと、永遠に着手できない。

5. 長文を「行動」に変えるプロンプト

募集要項、規約、役所の文章、病院の案内、PDFの資料。
世の中の文章、長すぎます。読む気を失わせる才能に満ちている。

有料ChatGPTの強さは、こういう時にファイルやPDFをそのまま投げやすいことです。

この文章/PDFを、
1. 中学生にも分かる日本語で3行要約
2. 私が今やるべきことを3つ
3. 締切や重要事項
4. 分からない用語の簡単な説明
に分けて整理してください。

さらに、こう続けてもいい。

私はADHDで長文が苦手です。
重要でない部分は削って、行動に必要な情報だけ残してください。

これは本当に便利です。
ただし、契約、医療、法律、金銭の重要事項は必ず自分でも確認してください。
AIは超優秀な下書き係ですが、最終責任者ではありません。

6. noteや発信の下書きを作るプロンプト

AIを使うと、発信がめちゃくちゃ楽になります。
ここで勘違いしてほしくないのは、全部を書かせる必要はないということです。

むしろ強いのは、

  • タイトル案
  • 導入文
  • 見出し構成
  • 言い換え
  • 途中で詰まった段落の叩き台

このへんです。

たとえばこう。

以下は私のメモです。note記事にしたいです。
対象読者: AIをあまり使ったことがない非エンジニア
目的: 読んだ後に有料ChatGPTを使いたくなること
トーン: 強めだが分かりやすい。偉そうすぎない。
出力:
- タイトル案を5つ
- 導入文
- 見出し構成
- 各見出しで何を書くか
- 締めの一文
[メモ]

さらに、自分の過去記事を2〜3本貼って、

この文体の特徴を抽出して、寄せて書いてください。
ただし、読みやすさは優先してください。

と頼むと、かなり「自分っぽい」下書きになります。

ADHDは、白紙に向かう瞬間が一番きつい。
でもAIが叩き台を出してくれると、編集モードに入れる。
ゼロから作るより、一を直す方が圧倒的に楽です。

7. 感情が荒れている時の整理プロンプト

ADHDは、感情の熱量が高いまま言葉に乗ることがあります。
それ自体は悪ではないけれど、勢いのまま送ると事故る。

そんな時は、送る前にAIをクッションにする。

私は今かなりイラついていて、勢いで返信しそうです。
以下の出来事について、
1. 事実
2. 私の解釈
3. 本当に伝えたい要望
4. 送ってもよい穏当な返信文
の順で整理してください。
条件:
- 私の感情を否定しない
- ただし攻撃的な表現は避ける
- 曖昧な決めつけは減らす

これ、かなり効きます。

ポイントは、AIに
「代わりにキレてもらう」のではなく、「熱を下げて整理してもらう」

8. 片付けを始めるプロンプト（画像ありだと強い）

片付けは、タスクの集合体です。
何から手をつけるか分からないと、脳がすぐ逃げます。

部屋の写真を撮って、こんなふうに頼めます。

この部屋の写真を見て、
15分で見た目が最も改善する順に片付け手順を出してください。
条件:
- 一度に1カテゴリだけ指示する
- 捨てる / 後で判断 / 元に戻す に分ける
- 私が迷わないようにする
- 最初の3分でやることを最初に書く

視覚情報をそのまま渡せるのは、本当に大きいです。
「説明する」コストが減るから。
ADHDは説明で疲れることも多いので、画像で済むのはありがたい。

ADHDがAIで失敗しやすい使い方

AIは強いですが、雑に使うと微妙になります。
よくある失敗はこの3つです。

1. 一発で完璧を求める

AIは対話型です。
1回目で80点が出たら、そこから修正すればいい。
最初から100点を狙わない。

使える追撃は、これです。

もっと短く
もっと初心者向けに
3ステップにして
例を2つつけて
一問ずつ聞いて
最初の5分だけに絞って

この追撃だけで、回答はかなり実用になります。

2. ふわっと頼みすぎる

「なんかいい感じにして」は、AIも困ります。
雑でいいけど、何に困っているかだけは入れる。

悪い例:

仕事を手伝って

良い例:

14時までに請求書送付、16時会議、返信2件があります。
今は眠くてやる気がなく、何から始めるか分かりません。
最初の15分でやることを1つだけ決めてください。

3. 大事な情報を鵜呑みにする

これは本当に注意です。
AIは間違えることがあります。

特に、

  • 契約
  • 税金
  • 医療
  • 法律
  • お金の判断

ここは必ず元の資料や専門家も確認してください。

AIは、あなたの判断を速くする道具ではある。
でも、判断責任そのものを肩代わりする道具ではないです。

結論：ADHDに必要なのは、根性論ではなく「外部化」だ

私は、ADHDに必要なのは根性ではないと思っています。

もちろん、根性がゼロでいいわけではない。
でも、毎回そこに頼るのは無理がある。

必要なのは、

  • 頭の中を外に出すこと
  • 順番を外に作ること
  • 最初の一歩を外に決めてもらうこと
  • 書けない時に叩き台を外から受け取ること

つまり、外部化です。

AIは、この外部化と相性が異常にいい。

しかも、コードなんて1行も書かなくていい。
難しい知識もいらない。
必要なのは、「今ここで困っていること」を、そのまま渡すことだけです。

私はもう、AIなしの生活には戻りたくありません。
それは「すごい文章が書けるから」だけではない。

  • 朝が始まりやすくなる
  • 返信が怖くなくなる
  • 長文の壁が低くなる
  • 白紙の前で固まりにくくなる
  • 頭の中が少し静かになる

こっちの方が大きい。

まだAIをほとんど使ったことがない人ほど、
まずは1か月、有料ChatGPTを生活に入れてみてほしいです。

最初に変わるのは、能力ではありません。
1日が始まる速さです。

それだけで、世界はかなりマシになります。
