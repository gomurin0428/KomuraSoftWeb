サービス

お客様のビジネスを支えるソフトウェア開発をお手伝いします。

Windows アプリ開発
デスクトップアプリケーション、業務システム、ツール開発など、Windows 向けソフトウェアの設計・開発を行います。
既存システムの改修・保守
既存システムの機能追加、バグ修正、パフォーマンス改善、モダナイゼーションなどをサポートします。
技術コンサルティング
技術選定、アーキテクチャ設計、コードレビューに加え、既存システムの改修方針整理、COM / ActiveX の保守・移行、32bit / 64bit の相互運用、不具合調査などの技術相談に対応します。

こんな技術相談に対応できます

  • COM / ActiveX / OCX の保守・移行
  • C# / .NET / WPF / WinForms の設計レビュー
  • 32bit / 64bit の相互運用や C++/CLI の整理
  • ハング、デッドロック、性能問題の切り分け
  • 既存 Windows アプリの改修方針の整理
  • 受託開発に入る前の技術的な壁打ち
