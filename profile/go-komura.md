## どんな案件で力を出しやすいか

代表の小村豪は、Windows ソフトウェアを中心に、既存資産が重く残る案件や、原因が見えにくい障害調査と相性の良い実務経験を積んできました。

特に次のような場面で価値を出しやすいです。

- COM / ActiveX / OCX を含む古い Windows 構成の整理
- C# / .NET、WPF / WinForms、C++/CLI、ネイティブ連携の境界設計
- 32bit / 64bit 相互運用を含む改修方針の整理
- 長期稼働後だけ発生するクラッシュ、停止、リークの調査
- ログと異常系テスト基盤を含めた再発防止の設計

## 技術的な見方の特徴

現象だけでなく、構成・責務・境界の置き方まで含めて問題を見る傾向があります。
そのため、単発の不具合調査だけでなく、調査しやすい構造に寄せるための設計相談にもつなげやすいのが特徴です。
