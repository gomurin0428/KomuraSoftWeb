## Why this page exists

When everything lives only on the homepage, people can still read it, but the entry points for specific intent become less obvious.
This page organizes the technical themes that KomuraSoft is asked about most often so visitors can start from the topic that matches their situation.

## Main topics

### ActiveX migration

This topic page is the entry point for deciding whether COM / ActiveX / OCX assets should be kept, wrapped, or replaced in stages.
It is useful when the real question is not "How do we rewrite everything?" but "How do we move safely from what we already have?"

### 32-bit / 64-bit interoperability

This topic page is the entry point for 32-bit / 64-bit boundaries, C++/CLI decisions, and native DLL integration strategy.
It is useful when modernization is blocked by technical boundaries rather than by business logic itself.
